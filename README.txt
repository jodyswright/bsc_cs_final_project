Code for the project is located in the submission folder

data_tools.py is a common module for loading the data

The remaining three parts are Jupyter notebooks consitsing of the following:

1. project-pre-processing.ipynb used for processing the data such that it is useable in an ML model 

2. project-testing-class.ipynb establishes the baseline and then runs the various keras optimzers

3. project-GA a bespoke genetic algorithm adapted to work as an hp optimizer

Source data are not included in this repository due to size
